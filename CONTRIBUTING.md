# Contributing

Thank you for your interest in Highlander Duelists!

This is a **private club project** maintained exclusively by the Highlander Duelists officer team at UC Riverside. We are **not accepting outside contributions** at this time.

## For Club Members

If you are a club officer or member with access, please coordinate directly with the project maintainer before making any changes.

- Do not push directly to `main` (currently we are on early development phase so you might see me doing a lot of main branch pushing 😅)
- Open an issue or message us on Discord before starting work
- All changes should go through a pull request

## Questions?

Reach out to us on our [Discord server](https://discord.gg/WHbpFPM2) or contact the club officers directly.
